
# Pandas
在計算機編程中，pandas是Python程式語言的用於數據操縱和分析的軟體庫。
## 導入pands
```python
import pandas as pd
```
## 可以導入Series和DataFrame，因為這兩個經常被用到
```python
from pandas import Series, DataFrame
```
## 1 Series 
### series是一個像數組一樣的一維序列，並伴有一個數組表示label，叫做index。創建一個series的方法也很簡單： 
```python
obj = pd.Series([4, 7, -5, 3])
obj
```
```
0    4
1    7
2   -5
3    3
dtype: int64
```
